# superheroes

Get superhero names

Inspired by [Sindre Sorhus's superheroes](https://github.com/sindresorhus/superheroes) 

## Install

``` 
go get -u github.com/fernandoporazzi/superheroes
```

## Usage

To return all superheroes from the list:

```go
superheroes.All()
```

To return a random superhero:

```go
superheroes.Random()
```
